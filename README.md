A Python application to manage movies, store them in a SQLite database, fetch poster images from the OMDb API, and generate a static HTML website.

## Features
- Add movies using OMDb API (title, year, rating, poster)
- Delete movies
- Update movie ratings
- List all stored movies
- Movie statistics (best, worst, average rating)
- Search movies
- Sort movies by rating
- Generate a website from a template
